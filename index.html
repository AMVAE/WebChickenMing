Django Setup Guide

Step-by-Step Instructions

python -m venv myvenv
myvenv\Scripts\activate
myvenv\Scripts\Activate.ps1
python -m pip install Django
django-admin startproject myproject
cd myproject
python manage.py runserver

python manage.py startapp myapp

Troubleshooting

If you get an error, run:
Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope CurrentUser

Settings Configuration

In settings.py:
- Add 'myapp' to INSTALLED_APPS
- Add templates: 'DIRS': [os.path.join(BASE_DIR, 'templates')]
- Add static files: STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]

URL Configuration

1. Create a new urls.py in myproject/myapp:

from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]
2. In myproject/urls.py, include the app URLs:
from django.urls import path, include

urlpatterns = [
    path('', include('myapp.urls')),
]

Views Configuration

In views.py add this function:

def home(request):
    return render(request, 'index.html')

Static Files Usage

Use these lines in your HTML template to load static files:

{% raw %}{% load static %}{% endraw %}
